# ✈️ Airplane Booking System (Java)

This is a *Console-based Application* developed during my *2nd Semester* (Programming Fundamentals) at *COMSATS University Islamabad, Sahiwal Campus*.

### 🛠️ Key Features
* *Dynamic Seat Mapping:* Uses a *2D Array (4x5)* to manage and display seat availability in real-time.
* *Data Persistence:* Integrated *File I/O* (using PrintWriter and Scanner) to save and load booking data from a .txt file.
* *Flight Selection:* Users can choose from 4 different routes with specific timings and prices.
* *Robust Input Validation:* Prevents the system from crashing due to invalid user inputs.

### 💻 Concepts Covered
* Multi-dimensional Arrays (2D)
* File Handling in Java
* Modular Programming (Methods)
* Conditional Logic & Loops
